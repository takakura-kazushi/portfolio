# Portfolio Site Project: "Minimalist Cockpit" Specification

## 1. コンセプト・世界観

- **テーマ**: 3次元空間の解析と情報の同期
- **ビジュアル**: 白背景を基調とした、清潔感のある未来的・SF的なミニマリズム
- **ターゲット**: 技術とデザインの両立を求めるエンジニア・デザイナー（インターン採用担当者）

## 2. 3Dメインビジュアル（Three.js）

- **オブジェクト**:
  - 形状: ぐねぐねと有機的に変形する多面体
  - カラー: ピンクと水色のパステル調グラデーション
  - 装飾: 表面に極細の白い骨格線（Wireframe）を重畳
- **背景**: 純白（#FFFFFF）
- **操作**:
  - PC: マウスドラッグによる回転、またはスクロール連動
  - Mobile: スワイプによる自由探索 ＋ 特定頂点へのマグネット吸着（オートスナップ）

## 3. UI/HUD（画面固定要素）

- **基本スタイル**: 極細のライン（Hairline）とタイポグラフィによるSF演出
- **配置**:
  - **左上（Top-Left）**: リアルタイムXYZ座標（垂直レイアウト）
    - `camera.position` の実数値を表示（小数点第3位まで）
  - **右上（Top-Right）**: 現在時刻（`HH:MM:SS:ms`）
  - **下部（Bottom）**: 0〜360°のメジャー（目盛り）
    - カメラの水平回転角（y-axis rotation）と連動して動く
  - **左辺・右辺（Sides）**: 極細の装飾ライン
    - 右辺にはカメラの移動速度に応じた「速度計」を検討（パフォーマンス次第）
  - **PC版限定**: 左側にマスク（遮蔽）要素を配置し、名前とタイトルを表示
- **フォント**: 未来的な等幅フォント（JetBrains Mono, IBM Plex Mono 等）

## 4. ナビゲーション & コンテンツ構造

- **情報階層**: ホーム + 3つの主要セクション（Profile / Works / Contact）
- **遷移トリガー（スキャン＆ロックオン）**:
  - 多面体の特定の3つの「頂点」に各セクションを紐付け
  - カメラが頂点に接近・合致した際にHUDが反応（ロックオン演出）
  - ロックオン状態でアクションを起こすと2Dページへ遷移
- **2Dページ**:
  - アニメーションにこだわったフラットな2Dレイアウト
  - 3D空間からのシームレスなズームイン/オーバーレイ遷移

## 5. 推奨技術スタック（実装フェーズ）

- **Framework**: Next.js (React)
- **3D Library**: React Three Fiber (R3F) + Drei
- **Animation**: GSAP (HUD・カメラワーク用) + Framer Motion (2Dページ用)
- **Styling**: Tailwind CSS
